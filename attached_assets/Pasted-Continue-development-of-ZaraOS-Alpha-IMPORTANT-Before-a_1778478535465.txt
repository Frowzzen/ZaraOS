Continue development of ZaraOS Alpha.

IMPORTANT: Before adding new features, make sure this project is not locked into Replit.

The code must be portable, GitHub-ready, Linux-ready, Tauri-ready, and eventually ready to be packaged into a custom Linux ISO for USB boot.

Do not use Replit-only services, Replit databases, Replit secrets, Replit auth, Replit-specific deployment assumptions, or anything that will break when exported to GitHub and run on a normal Linux machine.

PROJECT CONTEXT:
ZaraOS is a local-first, privacy-focused, AI-native operating environment. It will eventually be packaged into a bootable Linux-based USB OS, but right now we are building the software layer as a portable React + TypeScript app that can later become a Tauri desktop app.

ZaraOS must feel like a futuristic operating system, not a website.
